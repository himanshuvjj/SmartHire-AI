# Overview

SmartHire AI is an AI-powered Resume Screening and Candidate Tracking System designed to automate recruitment workflows using Machine Learning and NLP.

The system allows recruiters to upload resumes, calculate AI match scores, extract skills automatically, rank candidates, manage recruitment workflows, and visualize hiring analytics.

---

# Features

- Resume Upload System
- Resume Parsing
- AI Match Score Calculation
- Skill Extraction
- Candidate Ranking
- Analytics Dashboard
- Email Notification System
- Authentication System
- SQLite Database Integration
- Streamlit Multipage UI
